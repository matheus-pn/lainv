<h1> What's this? </h1>

Lainv is a simple text-based image viewer for the terminal enviroment

<h3>Features:</h3>

* 8-bit color
* zoom in/out

<h2>Example</h2>

  <h3>Lain!!</h3>
  
![Lain!!](https://i.imgur.com/scUcd41.png)

<h2> How to build </h2>

```
gcc main.c -lncurses -lm -o lainv
```

<h2> Usage </h2>

```
lainv image.jpg
```
